### Как устроена иерархия
```mermaid
  sequenceDiagram
      Средства Защиты (интерфейс)--> Информационная система(конкретный класс, реализующий интерфейс) ;
      Средства Защиты (интерфейс)--> Защита Рыцаря(абстрактный класс, реализующий интерфейс) ;
      Защита Рыцаря --> Защита тела рыцаря (абстрактный класс) ;
      Защита Рыцаря --> Щит (конкретный класс) ;
      Защита тела рыцаря --> Шлем (конкретный класс) ;
      Защита тела рыцаря --> Кольчуга (конкретный класс) ;
```
